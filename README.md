# health-check

> A GitHub App built with [Probot](https://github.com/probot/probot) that A Probot app

# Requirements

In order for nodegit to work, you need to have openssl installed localled

```sh
brew install openssl
```

## Setup

```sh
# Install dependencies
npm install

# Run typescript
npm run build

# Run the bot
npm start
```
